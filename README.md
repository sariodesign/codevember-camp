## 🚀 Quick Start

### Prerequisites

- Node.js 18+
- Supabase account (free tier)
- Google Cloud account (for Calendar API)
- Vercel account (for deployment)

### Local Development

```bash
# 1. Clone and setup
# with ssh key
git clone git@github.com:sariodesign/codevember-camp.git
# or without ssh key
git clone https://github.com/sariodesign/codevember-camp.git

cd codevember-camp
pnpm i

# 2. Environment setup
cp .env.example .env.local

# 3. Fill in your environment variables (see below)

# 4. Start development
pnpm dev

# 5. Open http://localhost:3000
```
